记华为云OBS的一次白嫖

12号早上一醒来，打开b站，映入眼帘的两个大字白嫖。

白嫖，我喜欢。

具体看了一眼视频

其实仔细分析一波，就是up在拉人头，不过对用户也有好处，属于双赢，我就注册了。

华为云叫OBS（Object Storage Service），阿里云叫OSS（Object Storage Service），腾讯云叫COS（Cloud Object Storage），商家显示出自家产品独一无二的特点，随便造名嘛，无所谓了。

用了优惠券之后，总价234，抵扣了200，又用红包，总共支付了33元。买了两年40G存储、50G/月下行流量，现在就用来做静态资源的加速了。

我自己的服务器是学生机，1M带宽，我之前的博客有分享大壁纸，网速125kb/s实在受不了。现在已经可以做到秒加载了。

除了用来加速静态资源，还可以用来做个人网盘，支持分享，也不错。反正这33块钱花得很值。

在配置obs绑定自定义域名的时候出了点问题，我工单反馈华为云，没屁用，好在群里有个大佬，分分钟帮我解决了。
